Lab 1 的 selfcheck 腳本使用方式

  1. 下載該 bash script
    $ wget https://concise.github.io/ntueeeslabfall2015-lab1/eslab1-selfcheck
  2. 假設你即將要繳交的檔案是 some/path/to/ESLab1_Team00.zip
  3. 執行該 bash script 協助你做檢查
    $ bash eslab1-selfcheck some/path/to/ESLab1_Team00.zip
  4. 當腳本輸出「 * Everything seems to be OK! 」時，表示你通過了基本檢查。

  5. 附註：要做出一個可通過此測試的壓縮檔應該是非常簡單。舉例來說， 
    $ cd /tmp
$ mkdir ESLab2_Team01
$ touch ESLab2_Team01/report.pdf
$ zip -r ESLab2_Team01.zip ESLab2_Team01
$ wget https://concise.github.io/ntueeeslabfall2015-lab1/eslab2-selfcheck
$ bash eslab2-selfcheck ESLab2_Team01.zip
